# 🚀 Portfolio Fullstack - Mehdi

Portfolio moderne conçu avec **React**, **TypeScript** et **Framer Motion**. Ce projet met en avant mes compétences en développement Fullstack et Cloud, avec un design néon immersif et responsive.

## 🛠️ Stack Technique

- **Frontend** : React 18, Vite, TypeScript
- **Animations** : Framer Motion (effets de flottement et carrousel infini)
- **Design** : CSS moderne avec variables dynamiques
- **Déploiement** : Vercel / AWS Amplify

## ✨ Fonctionnalités

- **Hero Section** : Animations interactives avec images néon (Café & Fusée).
- **About Section** : Présentation de mon expertise Java/Spring Boot & React.
- **Tools Carousel** : Carrousel infini affichant mes outils favoris (AWS, Docker, GitHub, etc.).
- **Responsive Design** : Optimisé pour tous les écrans (Mobile, Tablette, Desktop).
